# v110 만족도조사 속도/안정화 점검

- 만족도조사 테스트 링크 생성 시 구글시트 백업을 기다리지 않고 Supabase 원본 저장 후 즉시 링크를 열도록 수정
- 테스트 문자 발송도 Supabase 저장 → 문자 발송 → Supabase 상태 갱신 순서로 빠르게 처리하고, 구글시트는 백그라운드 자동 백업으로 분리
- 설문 링크 조회 API에서 구글시트 조회 fallback을 제거하고 Supabase 원본 조회만 사용해 로딩 지연 방지
- 설문 응답 제출 API는 Supabase 저장 성공 시 즉시 성공 응답을 주고, 구글시트 백업은 짧은 제한 시간 안에서만 시도해 제출 화면이 오래 멈추지 않도록 수정
- 만족도조사현황 화면은 최근 300건 우선 표시로 렌더링 부담 완화
- 만족도조사 응답/발송 기록 삭제도 Supabase 우선 저장 후 구글시트 백업은 백그라운드 처리
- 캐시 버전 v110 반영
