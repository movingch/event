# v101 - Supabase 원본 전환 후 구글시트 불러오기 문구/동작 정리

- Supabase 강제 이전 이후에도 남아 있던 구글시트/구글드라이브 최초 불러오기 안내 문구를 제거했습니다.
- 앱 시작 화면을 Supabase 원본 불러오기 기준으로 정리했습니다.
- 구글시트는 원본이 아니라 Supabase 저장 성공 후 따라오는 자동 백업/출력용으로 명확히 정리했습니다.
- 구글시트 백업 전에 구글시트 데이터를 다시 읽어와 현재 화면을 덮어쓰는 동작을 차단했습니다.
- 기존 UI의 '구글시트 자동동기화' 문구를 '구글시트 자동백업'으로 변경했습니다.
