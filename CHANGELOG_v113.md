# v113 관리자 메뉴/만족도 초기화/구글백업 안정화

- 관리자 사이드 메뉴를 버튼 의존 방식에서 해시 링크 기반으로 되돌려 클릭 안정성을 높였습니다.
- 일반관리자/마스타관리자 로그인 상태에서 허용된 메뉴가 항상 열리도록 라우팅을 보강했습니다.
- 마스타관리자 만족도조사 메뉴에 수신·문자발송내역 초기화, 응답현황 초기화 버튼을 추가했습니다.
- 구글시트 자동백업을 직렬 큐로 처리하여 백업 중 새 변경이 생기면 최신 상태를 이어서 다시 백업하도록 했습니다.
- Apps Script 백업 대기 시간을 45초로 늘려 느린 구글시트 저장 때문에 누락되는 일을 줄였습니다.
