# v77 만족도조사 시스템 반영

- ADMIN 메뉴에 `만족도조사` 메뉴를 추가했습니다. 위치는 `백업·연동` 바로 위입니다.
- 관리자 로그인/라우팅 구조는 변경하지 않고 v76 안정 구조를 유지했습니다.
- 만족도조사 전체 ON/OFF, 문자 자동발송 ON/OFF, 발송 지연 시간, 응답 마감일, 중복응답 방지, 문자 템플릿을 관리할 수 있습니다.
- 설문 문항 기본 6개와 문항 수정 화면을 추가했습니다.
- 회차별 설문 사용/자동문자 설정을 추가했습니다.
- 참석자에게 발송되는 설문 링크는 토큰 방식으로 구성되며, 개인정보를 URL에 직접 노출하지 않습니다.
- `survey.html` 설문 응답 페이지와 `/api/survey` 프록시 API를 추가했습니다.
- Google Apps Script v77은 기존 디자인 탭에 더해 `만족도_설정`, `만족도_문항`, `만족도_응답`, `만족도_통계`, `만족도_문자발송기록` 탭을 디자인된 형식으로 생성/갱신합니다.
