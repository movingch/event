# v84 통합본

v81, v82, v83 수정 사항을 하나로 합친 통합 업로드본입니다.

## 포함 내용

### v81
- 모바일 개막작 노출 설정 적용 보강
- 모바일 ADMIN 신청자 명단 최적화
- 모바일 STAFF 신청자 명단 최적화

### v82
- 신청/참석/미참석 3가지 상태 체계 통일
- 신청자목록, 참석여부 필터, 상영관별 통계, 영화별 신청·참석 현황에 동일 기준 적용
- 표와 버튼 여백 압축
- 구글시트 전송 데이터 상태 기준 정리

### v83
- 모바일에서 개막작 노출 기간 설정이 남아 보이던 문제 재수정
- 데스크톱, 모바일, 영화카드 목록의 개막작 노출 판단 통일

## Apps Script
- 구글시트 상태 체계 정리는 v82 Apps Script 기준입니다.
- 이미 GOOGLE_APPS_SCRIPT_CODE_v82.gs로 교체 및 새 버전 배포했다면 추가 변경은 필요 없습니다.
