# v127 만족도조사 안정성 개선

- 설문 제출 시 유효한 토큰, 활성화 상태, 응답기한, 중복 설정을 서버에서 확인합니다.
- 필수문항, 복수선택, 점수 범위, 객관식 선택지를 서버와 화면에서 검증합니다.
- 동시 응답 저장 충돌을 감지하고 재시도하도록 개선했습니다.
- 관리자가 다른 내용을 저장하는 순간에도 새 설문 응답과 응답완료 상태가 사라지지 않도록 병합합니다.
- 테스트 응답과 테스트 발송을 실제 통계에서 제외했습니다.
- 참석자 통계를 예약 건수가 아닌 실제 참석 인원으로 계산합니다.
- 설정된 별점 문항을 기준으로 통계 카드를 자동 구성합니다.
- 구글시트 설문 응답 열을 정렬하고 신청인원과 전체 응답 JSON을 보존합니다.
- 설문 화면에 응답 마감일과 응답 자료 이용 안내를 표시합니다.
