# v14 변경사항

- 모든 상영 회차의 최초 스태프 비밀번호를 `0909`로 통일
- 스태프 로그인 시 선택한 담당 회차 한 곳의 신청자 명단만 접근하도록 제한
- 스태프가 현재 비밀번호 확인 후 자신의 회차 비밀번호를 직접 변경하는 기능 추가
- 신청자 명단을 상영관, 상영시간, 신청자 이름 순으로 정렬
- 신청자 명단의 문자 상태 열과 문자발송 버튼 제거
- 인쇄용 참석 칸에서 `참석 체크` 문구 제거
- 신청자 칸 너비 및 인쇄 표의 줄·칸 간격 최적화
- A4 가로 인쇄와 회차별 페이지 분리 적용

## 보안 참고
현재 버전은 브라우저 localStorage 기반이므로 서버 인증과 같은 강한 보안은 아닙니다. 실제 여러 기기에서 신청자 명단을 공유하고 담당자별 권한을 안전하게 적용하려면 중앙 데이터베이스와 서버 인증이 필요합니다.
