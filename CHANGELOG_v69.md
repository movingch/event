# v69 변경사항

- 관리자 메뉴 버튼/링크가 작동하지 않는 상황을 우회하기 위해 독립 페이지 `backup.html`을 추가했습니다.
- `https://cine-event.vercel.app/backup.html?v=69`에서 관리자 화면을 거치지 않고 구글시트 저장, JSON 백업, 신청자/통계 엑셀 저장을 실행할 수 있습니다.
- `backup.html`은 같은 도메인의 localStorage를 직접 읽어서 현재 브라우저의 신청자 수를 표시하고, Vercel API 프록시를 통해 Apps Script로 저장합니다.
- 관리자 화면 상단에도 백업·연동 즉시 실행 패널과 별도 백업페이지 링크를 추가했습니다.
- 앱 상단 메뉴에 `백업 바로가기` 링크를 추가했습니다.
