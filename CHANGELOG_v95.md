# v95 - 구글시트 원본 자동 동기화

- 구글시트를 원본 데이터로 사용하도록 변경했습니다.
- 모든 브라우저, 모든 관리자, 모든 스태프, 모바일/PC에서 페이지가 열릴 때 구글시트 데이터를 자동으로 불러옵니다.
- 60초마다 구글시트 최신 데이터를 다시 확인합니다.
- 브라우저별 localStorage 데이터 차이 때문에 서로 다른 데이터가 보이는 문제를 줄였습니다.
- Vercel API에 구글시트 읽기 프록시를 추가했습니다.
- Apps Script v95는 `action=exportData`로 신청자현황, 상영관영화, 만족도 응답/문자 기록을 웹앱에 돌려줍니다.
