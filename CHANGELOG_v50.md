# v50 신청자 명단 날짜 필터·예약문자 일괄발송

- 관리자 모드 신청자 명단 날짜 필터에 `전체 날짜` 선택과 날짜 직접 선택을 함께 제공합니다.
- `예약문자전송하기` 버튼을 인쇄용 명단 버튼 앞에 추가했습니다.
- 문자 발송 모드에서 신청자 명단 앞 체크박스로 대상자를 선택할 수 있습니다.
- 선택 대상에게 `예약관련문자 보내기`로 기존 예약 완료 문자를 재발송할 수 있습니다.
- 선택 대상에게 `별도안내문자 보내기`로 운영진이 직접 작성한 안내문자를 발송할 수 있습니다.
- SENS API 함수가 별도 안내문자(`kind: notice`)를 처리하도록 확장했습니다.
