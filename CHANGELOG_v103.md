# v103 - Supabase 표시 문구 정리 및 구글시트 백업 보강

- 앱 시작 로딩 문구를 `최신 신청자 데이터`에서 `Supabase 원본 데이터`로 명확히 변경했습니다.
- 구글시트를 원본처럼 불러오는 동작은 계속 차단하고, Supabase 원본만 읽도록 유지했습니다.
- Supabase 저장 후 구글시트 백업이 서버 환경변수 문제로 건너뛰거나 실패할 경우 브라우저에서 한 번 더 백업 요청하도록 보강했습니다.
- 서버 API에도 기본 Apps Script 백업 URL fallback을 추가해 `GOOGLE_APPS_SCRIPT_URL` 누락 시에도 백업이 시도되도록 했습니다.
- 구글시트 백업 실패 응답을 더 명확히 감지하도록 보강했습니다.
