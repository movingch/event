# v35 수정 내역

- 메인 페이지의 개막작 신청 카드 제거
- “개막식 사전신청” 문구를 “개막작 신청”으로 변경
- 신청자 명단 표의 영화/시간, 신청자, 예약번호, 티켓/좌석, 인원 열 간격 축소
- 상세통계의 “영화별 신청·참석률”을 “영화별 통계”로 변경하고 상영관별 통계와 동일한 표 양식 적용
- 신규 신청은 정원과 무관하게 즉시 확정 처리
- 신청자 명단의 확정/대기 상태 변경 버튼 제거
- 참석/취소 버튼을 현장 참석 여부 확인용으로 정리하고 취소 후 바로 참석 처리 가능하도록 변경
- 참석 인원 입력 시 신청 인원 칸도 실제 입력 인원으로 자동 반영
- 신청자 수정 버튼을 누르면 주요 신청 항목을 한 번에 수정하는 화면으로 변경
