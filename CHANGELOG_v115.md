# v115 관리자 라우트 / 구글 백업 최종 보강

- /admin 일반관리자, /adminor 마스타관리자 라우트를 분리하고 관리자 내부 메뉴 hash를 #/stats, #/surveyView 방식으로 단순화했습니다.
- 과거 #/admin/stats 주소도 호환되도록 했습니다.
- 상세통계, 만족도조사현황 렌더링 오류가 나도 메뉴 화면이 완전히 죽지 않도록 안전 렌더링을 추가했습니다.
- 강제백업과 자동백업이 같은 서버 백업 API(/api/supabase-google-backup)를 우선 사용하도록 고정했습니다.
- 새 Apps Script URL을 기본값으로 반영하고, 기존 오래된 URL이 브라우저에 남아 있으면 새 URL로 교체합니다.
- 자동 백업 예약 간격을 줄여 Supabase 저장 직후 더 빨리 구글시트 백업이 따라가도록 했습니다.
