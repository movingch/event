# v107 - 만족도조사현황 / 테스트 응답 판정 보강

- 일반관리자 메뉴명을 `만족도조사보기`에서 `만족도조사현황`으로 변경했습니다.
- 일반관리자도 만족도조사 수신/발송 명단, 응답 목록, 실제 응답 내용을 볼 수 있도록 유지했습니다.
- 테스트 설문 링크/문자 생성 시 발송 기록을 Supabase 원본에도 즉시 저장하도록 보강했습니다.

- 테스트 응답도 `TEST`로 구분하면서 만족도 통계와 응답률에 반영되도록 했습니다.
- 만족도 응답/발송 기록 삭제 후 Supabase와 구글시트 백업에 즉시 반영되도록 했습니다.
