# v70 자동 동기화/관리자 로그인/백업 노출 정리

- 메인 헤더에서 `백업 바로가기` 버튼을 완전히 제거했습니다.
- 백업 페이지 링크는 관리자 화면 내부에만 남겼습니다.
- 관리자 PIN 저장값이 꼬여도 기본 PIN `0909`로 로그인되도록 복구 경로를 추가했습니다.
- 구글시트 연동 URL을 저장하면 자동 동기화가 즉시 켜지도록 보강했습니다.
- 기존에 URL만 저장되어 있고 자동동기화 플래그가 없는 브라우저도 자동 동기화가 켜진 상태로 인식하도록 수정했습니다.
- 별도 백업 페이지에서 구글시트 저장에 성공하면 자동 동기화 플래그도 함께 저장되도록 수정했습니다.
- app.js, api/google-drive-sync.js 문법 검사를 통과했습니다.
