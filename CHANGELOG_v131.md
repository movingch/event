# v131 변경사항

- 상영관 정원 도달 시 이후 신청을 자동으로 `대기` 상태로 접수
- 정원 도달 화면에 `현재 이 상영관은 신청마감이 되었습니다. 대기상태로 신청을 받겠습니다.` 안내 추가
- 대기 신청 완료창과 예약 문자에 대기 상태 및 후속 안내 반영
- 네이버 SENS 예약 문자에서 `대기` 상태를 구분해 자동 안내
- 관리자 신청자 관리에서 대기 신청을 확정으로 전환하는 기능 추가
- 잔여 정원이 부족할 때 대기 신청의 확정 전환 차단
- 영화 신청 페이지에 날짜별 제9회 상영시간표 추가
- 시간표의 영화를 누르면 영화 소개와 관람 신청 팝업 연결
- 영화 소개에 상영일시, 장소, 소개글, GV, 진행자, 정원, 확정·대기 현황 표시
- 동일 이름 또는 전화번호의 같은 영화·겹치는 시간·시작 시각 1시간 이내 중복 예약 차단
- 동일 이름 또는 전화번호 기준 최대 5회까지만 신청 허용
- 신청자 관리에 개인별 신청·확정·대기·참석·미참석 횟수와 신청 영화 현황 추가
- 각 신청자 행에 누적 신청·참석·대기 횟수 표시
