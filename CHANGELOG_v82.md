# v82 - 신청/참석/미참석 통일 및 표 압축

- 신청자 목록 참석여부 필터를 `신청 / 참석 / 미참석` 3종으로 통일했습니다.
- 신청자 목록, 상영관별 통계, 영화별 신청·참석 현황, 날짜별 통계에서 `취소` 상태 표시를 제거하고 `미참석` 기준으로 정리했습니다.
- 기존 `취소`로 저장된 신청 데이터는 화면/동기화 시 `미참석`으로 표시되도록 보정했습니다.
- 관리자·스태프 신청자 명단 및 통계표의 셀 여백, 글자 크기, 버튼 크기를 줄여 모바일 가로폭 부담을 낮췄습니다.
- 모바일 신청자 목록에서 예약번호/신청일/메모 열을 숨기고 핵심 출석체크 정보 중심으로 보이도록 정리했습니다.
- 구글시트 Apps Script v82에서 신청자현황 드롭다운과 통계 컬럼도 `신청 / 참석 / 미참석` 기준으로 맞췄습니다.
