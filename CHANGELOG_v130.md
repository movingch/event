# v130 변경사항

- 노션 `웹앱 수정 자료 정리`의 관리자 화면 수정사항 반영
- 개막작 현황 4개 카드를 항상 한 줄로 유지
- 운영요약 현황 5개 카드를 항상 한 줄로 유지
- 만족도조사 상태 4개 카드를 항상 한 줄로 유지
- 후원자 집계 현황 4개 카드를 항상 한 줄로 유지
- 좁은 화면에서는 카드가 줄바꿈되지 않고 가로 스크롤되도록 처리
- 만족도조사·자동문자 상태를 카드 클릭으로 바로 ON/OFF 전환
- 만족도조사 ON은 빨간색, OFF는 검정색으로 표시
- 전체 설정의 중복 ON/OFF 체크박스 제거
- 만족도조사현황의 중복 응답 건수 배지 제거
