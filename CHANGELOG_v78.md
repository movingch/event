# v78 신청자/스태프 출석체크 및 티켓·좌석 항목 정리

- 신청자 명단의 신청자 이름 옆 기본 표시를 `신청` 배지로 변경했습니다.
- 참석 체크 후에는 `참석`, 미참석 체크 후에는 `미참석` 배지로 표시됩니다.
- ADMIN 신청자 명단과 STAFF 신청자 관리 화면 모두 동일하게 적용됩니다.
- 관리 칸의 버튼을 `참석 / 미참석`으로 정리했습니다.
- 신청자 상세 관리 패널의 설명 문구를 삭제하고, 버튼을 중앙 정렬·대형화했습니다.
- 신청자 명단의 `티켓/좌석` 컬럼을 삭제하고 표 간격을 재조정했습니다.
- 수정 창에서 티켓 구분/좌석 입력 항목을 제거했습니다.
- 엑셀/구글시트 전송 자료에서 티켓·좌석 항목을 제거했습니다.
- Google Apps Script v78에서 신청자현황, 출력용, 통계, 상영관영화, 만족도_응답 탭의 티켓·좌석 관련 컬럼을 정리했습니다.
- 관리자 로그인/라우팅 구조는 변경하지 않았습니다.
