# v140 — 마스타관리자 후원자 관리

- 마스타관리자 전용 사이드 메뉴에 `후원자 관리`를 추가했습니다.
- 후원 신청 건수, 예정액, 감사문자 상태와 전체 후원자 목록을 한 화면에서 확인할 수 있습니다.
- 후원자 명단을 엑셀 파일로 저장할 수 있습니다.
- 후원자 목록 전체 초기화 기능에 2단계 확인과 마스타관리자 권한 검사를 적용했습니다.
- 초기화 성공 시 Supabase 원본을 먼저 저장하고 구글시트 `후원자현황` 백업도 갱신합니다.
- Supabase 저장에 실패하면 브라우저의 기존 후원자 목록을 자동 복구합니다.
- 후원자 초기화는 영화 관람 신청자와 상영정보를 삭제하지 않습니다.
