# v76 - Google Sheet 디자인 정리 버전

- ADMIN 관련 화면 구조는 건드리지 않고 안정 버전(v75)을 유지했습니다.
- Google Apps Script v76을 추가했습니다.
- 기존 구글시트 탭을 보고서형 디자인으로 정리합니다.
  - 신청자현황: 제목/요약/No/필터/색상/드롭다운/전화번호 서식
  - 신청자현황_출력용: 현장 출력용 간소화 명단 신규 생성
  - 통계: 상단 요약 카드 + 영화별 통계표 + 색상 강조
  - 상영관영화: 일정표형 디자인 + 상태 드롭다운 + 잔여석 강조
  - 저장기록: No/상태/필터/색상 적용
- 앱 스크립트가 저장 시 탭 디자인을 함께 적용합니다.
- index.html의 app.js 캐시 버전을 v76으로 올렸습니다.
