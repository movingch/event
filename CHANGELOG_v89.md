# v89 관리자 저장·출력 버튼 통일

- 운영요약, 신청자명단, 상세통계에서 공통으로 쓰는 `통계엑셀저장`, `신청자엑셀저장`, `인쇄PDF저장`, `로그아웃` 버튼을 관리자 상단 줄에 통일 배치했습니다.
- 로그아웃 버튼은 항상 마지막에 오도록 정리했습니다.
- 각 화면 안에 중복으로 있던 저장·출력 버튼을 정리했습니다.
- 통계/신청자 엑셀 저장 파일을 CSV 대신 Excel에서 바로 열 수 있는 `.xls` 형식의 정돈된 표 양식으로 저장하도록 바꿨습니다.
- 엑셀 저장 파일에 제목, 생성시각, 헤더 색상, 테두리, 여백, 줄 간격을 넣어 출력·공유용으로 보기 좋게 정리했습니다.
- 인쇄/PDF 저장은 기존 A4 출력 최적화 방식을 유지합니다.
