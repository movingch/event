# v128 시니어 친화 설문·최고관리자 편집 강화

- 설문 제출 버튼을 화면 너비에 맞는 72~76px 높이의 큰 버튼으로 변경했습니다.
- 질문, 선택지, 체크 영역과 입력 글자를 크게 하고 키보드 포커스를 선명하게 표시합니다.
- 최고관리자가 설문 제목, 시작 안내, 개인정보 안내, 제출 완료 문구를 수정할 수 있습니다.
- 설문 문항 추가, 복제, 삭제, 위·아래 순서 이동 기능을 추가했습니다.
- 최고관리자에서 관객용 설문 화면을 바로 미리 볼 수 있습니다.
- 새 설문 문구도 Supabase 원본과 구글시트 설정 탭에 백업합니다.
