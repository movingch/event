# v25 변경사항

- 후원 신청 후 감사 문자 발송 요청과 함께 이체 안내 화면으로 이동
- 이체 화면에 금액, 입금자명, 은행, 계좌번호, 예금주 및 복사 버튼 추가
- 실제 후원 계좌는 `app.js` 상단의 `DONATION_BANK_NAME`, `DONATION_ACCOUNT_NUMBER`, `DONATION_ACCOUNT_HOLDER`에 입력
- 예약번호 표시를 상영관 첫 글자-세 자리 숫자 형식으로 변경 (예: 동-001)
- 내부 데이터 식별자는 기존 `id`를 유지하여 관리 기능 충돌 방지
- Node.js 엔진을 22.x로 고정
