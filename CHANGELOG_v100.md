# v100 - Supabase 강제 이전 버튼 추가

- 마스타관리자 → 백업·연동에 `현재 브라우저 데이터를 Supabase로 강제 이전` 버튼을 추가했습니다.
- Supabase `festival_state`가 `{}`일 때 실제 신청자 데이터가 보이는 주 컴퓨터에서 버튼을 눌러 원본 데이터를 강제로 채울 수 있습니다.
- 앱 시작 시 구글시트 강제 불러오기 보조 경로를 제거했습니다.
- 운영 기준을 `Supabase = 원본`, `구글시트 = 현재 구조 그대로 자동 백업`으로 정리했습니다.
- Apps Script 교체는 필요하지 않습니다.
