# v139 변경사항

- 로컬 서버의 ADMIN 링크를 `adminPreview=1` 관리자 미리보기 주소로 분기
- 배포 환경에서는 기존 `/admin` 일반관리자 주소 유지
- 로컬 관리자·최고관리자 화면의 홈 로고가 미리보기 쿼리를 제거하고 실제 홈으로 이동하도록 수정
- 별도 백업페이지의 관리자 복귀 링크를 최고관리자 백업 메뉴로 연결
- 로컬에서는 `masterPreview=1`, 배포 환경에서는 `/adminor` 주소를 사용하도록 자동 분기
- 공개·스태프·관리자 링크 최종 점검 버전 갱신
- 관리자 경로에서 영화 신청·후원·STAFF 메뉴가 관리자 내부 해시로 잘못 해석되던 문제 수정
- 개막작 관객 화면의 `관리자 설정`과 관리자 개막작 화면의 `관객 화면 보기`를 올바른 상호 이동 주소로 수정
