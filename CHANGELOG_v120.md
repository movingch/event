# v120 후원 문구·동의서·전화번호·일반관리자 계정

- 메인과 후원 페이지의 후원 요청 안내문을 새 문구로 교체했습니다.
- 영화 관람 신청서에 개인정보 수집·이용 및 초상권 사용 동의서를 추가하고 기본 체크·필수 동의로 설정했습니다.
- 동의 여부, 동의 시각, 동의서 버전을 예약 원본과 신청자 백업 자료에 함께 저장합니다.
- Google Apps Script가 모든 시트 탭의 연락처·전화번호·휴대폰 열을 찾아 전화번호 표시를 하이픈 형식으로 통일합니다.
- 마스타관리자가 일반관리자를 이름·비밀번호 계정으로 추가, 수정, 삭제할 수 있도록 변경했습니다.
- 기존 단일 일반관리자 PIN은 `일반관리자` 이름의 임시 계정으로 자동 이전됩니다.
- 캐시 버전을 120으로 올렸습니다.
- 로컬 `vercel dev` 실행을 재귀 호출하던 package 스크립트를 제거했습니다.
