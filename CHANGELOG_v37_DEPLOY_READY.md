# v37 Deploy Ready

이 패키지는 터미널 없이 GitHub 웹 업로드로 배포할 수 있도록 정리한 수정본입니다.

## 포함된 기능

1. 메인 메뉴에서 개막작 신청 버튼 삭제
2. 스태프 메뉴를 `STAFF`로 변경하고 작게 표시
3. 관리자 메뉴를 `ADMIN`으로 변경하고 작게 표시
4. STAFF 화면의 `담당 상영회차` 문구를 `담당 상영 영화`로 변경
5. ADMIN 모드에 `STAFF 관리` 탭 추가
6. 영화별 STAFF 상태 표시: `STAFF 없음` / `STAFF 있음`
7. STAFF가 비밀번호를 입력하고 들어오면 해당 영화가 자동으로 `STAFF 있음` 처리
8. ADMIN에서 영화별 STAFF 비밀번호 저장 가능
9. ADMIN에서 STAFF 상태를 `없음`으로 초기화 가능
